# Gua

Windows 平台上的快速启动器。按下热键呼出搜索框，输入识别码快速打开程序、文件、网站，或输入搜索引擎识别码加关键词快速搜索。

## 功能

- **热键呼出**：默认 `Alt+Space`，可自定义任意组合键
- **快速启动**：输入识别码回车打开程序、文件、文件夹、网址、系统命令
- **快捷搜索**：`识别码 关键词` 自动拼接搜索引擎 URL
- **拼音搜索**：输入拼音匹配中文条目，如 `bili` → 哔哩哔哩
- **拼音首字母**：输入 `fq` 匹配「番茄小说」
- **模糊匹配**：输入字符按顺序出现在条目名中即可匹配

## 快速开始

1. 运行 `gua.exe`，程序自动隐藏到系统托盘
2. 按热键（默认 `Alt+Space`）呼出搜索框
3. 输入识别码，按回车执行

## 设置面板

托盘右键菜单 → "设置" 打开设置面板，4 个 tab：

### 识别码

value 支持以下格式：

| 格式 | 说明 | 示例 |
| --- | --- | --- |
| 网址 | 填入 URL，回车后浏览器打开 | `https://www.bilibili.com` |
| 搜索 | 搜索引擎基础链接，URL 末尾带 `=` | `https://www.baidu.com/s?wd=` |
| 文档 | 填入文件路径，按默认应用打开 | `E:\readme.txt` |
| 文件夹 | 填入文件夹路径，回车打开资源管理器 | `E:\我的项目\gua` |
| 程序 | 填入 `.exe` 路径，直接启动 | `E:\WezTerm\wezterm-gui.exe` |
| 脚本文件 | 填入 `.bat`/`.py`/`.ps1` 等脚本路径，由关联解释器运行 | `E:\scripts\sync.bat` |
| 命令 | 路径用反引号 `` ` `` 包裹，按命令行执行 | `` `explorer shell:MyComputerFolder` `` |

- 控制台命令会弹终端窗口，配合 `cmd /k` 可保持窗口不关

  例如：`` `ipconfig` `` 弹终端窗口运行后自动关闭，`` `cmd /k ipconfig` `` 弹终端窗口并保持显示

### 通用

| 设置项 | 说明 |
| --- | --- |
| 热键 | 点击按钮，按下新组合键即可绑定 |
| 失去焦点自动隐藏 | 窗口失焦时自动隐藏 |
| 模糊匹配 | 字符按顺序出现即匹配，≥2 字符触发 |
| 拼音搜索 | 中文条目自动转拼音匹配 |
| 区分大小写 | 匹配识别码时是否区分大小写 |
| 黑名单程序 | 前台窗口是黑名单程序时，热键不响应 |
| 多音字追加 | 如 `茄(qie)`，`fanqie` 和 `fanjia` 都能匹配到"番茄" |

### 外观

| 设置项 | 说明 |
| --- | --- |
| 背景色 / 输入框色 / 高亮色 / 文字色 | RGB 十六进制，如 `#1E1E1E` |
| 字体 | 从下拉列表选择，`fonts/` 目录放入字体后点「刷新」即可出现 |
| 字号 | 搜索框文字字号 |
| 状态栏字号 | 底部「第 N 条/共 M 条」的字号 |
| 透明度 | 0（全透明）\~ 255（不透明） |
| 圆角大小 | 窗口圆角像素值 |
| 水平位置 | 0（左）\~ 100（右），50 为居中 |
| 垂直位置 | 0（上）\~ 100（下），50 为居中 |
| 宽度 | 窗口宽度（像素） |
| 最大显示限制 | 下拉列表最多显示的条目数 |

## 搜索匹配

输入识别码时按以下层级匹配，精确结果排在最前：

| 层级 | 匹配方式 | 示例（输入 `bili`） |
| --- | --- | --- |
| 1 | 精确匹配（key） | `bili` |
| 2 | 前缀匹配（key） | `bilibili` |
| 3 | 子串匹配（key） | `abcli` |
| 4 | 拼音前缀 | 哔哩哔哩（拼音 `bilibili` 前缀） |
| 5 | 拼音子串 | 哔（拼音 `bi` 子串） |
| 6 | 拼音首字母 | `fq` → 番茄（fanqie） |
| 7 | 模糊匹配（key） | `BiliHub` 字符按顺序出现 |

## 热键格式

```plaintext
修饰键 + 修饰键 + ... + 按键
```

- 修饰键：`Alt` `Ctrl` `Shift` `Win`
- 按键：`A~Z` `0~9` `F1~F24` `Space` `Enter` `Esc` `Tab` `Backspace` `Delete` `Insert` `Home` `End` `PageUp` `PageDown` 方向键 符号键
- 示例：`Alt+Space` `Ctrl+J` `Ctrl+Shift+F` `Alt+F1`

## 从源码构建

```bash
cargo build --release
```
